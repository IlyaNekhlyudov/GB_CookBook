в сборку накатил bootstrap, vue, регистрацию/аутентификацию у вас должен быть установлен Node.js чтобы пользоваться командой npm. npm install && npm run dev - установка зависимостей проекта после пулл реквеста с гита. npm run dev - еще раз повторить в консоли команду для сборки css/js файлов php artisan serve - запуск проекта (в консоли появится локальный адрес). после создания базы, надо либо поднять миграции (из папки проекта в консоли php artisan migrate), либо залить бд из папки sql базу данных у себя назвал gb_cookbook, collation = utf8mb4_general_ci (нужно настроить подключение к базе в файле .env в корне проекта)
