заходите в папку с проектом, в консоли(IDE или системной) делаете команду git clone https://github.com/Elvis97rus/GB_CookBook.git у вас должен быть установлен Node.js чтобы пользоваться командой npm. проверить есть он или нет можно в консоли(IDE или системной) командой node -v после git clone: npm install && npm run dev - установка зависимостей проекта после пулл реквеста с гита. npm run dev - еще раз повторить в консоли команду для сборки css/js файлов php artisan serve - запуск проекта (в консоли появится локальный адрес). после создания базы, надо либо поднять миграции (из папки проекта в консоли php artisan migrate), либо залить бд из папки sql базу данных у себя назвал gb_cookbook, collation = utf8mb4_general_ci (нужно настроить подключение к базе в файле .env в корне проекта) в сборку накатил bootstrap, vue, регистрацию/аутентификацию ***************************************************************************** Git ***************************************************************************** Прежде чем начинать работать сделай собственную ветку командой git checkout -b branch_name убедитесь что вы находитесь в ней командой git status После этого получите обновления/изменения в других ветках, без добавления в вашу командой git fetch Чтобы обновить локальную ветку, после получения обновлений других веток, надо добавить изменения командой git merge ветка_откуда_берем/ветка_куда_закидываем После завершения работы над какой то частю проекта ветка того кто делал мерджится с main-веткой, после этого всей команде надо обновить основную ветку командой git pull гит_ветка локальная_ветка *если вы делали изменения в ветке в которую делаете pull-request, убедитесь, что вы сделали commit последних изменений перед pull-request-ом, чтобы не потерять изменения*
